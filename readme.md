# dir-blast 

一个 CTF 快速扫描目录工具

## 功能

- 使用字典文件对目标 URL 进行目录暴力破解
- 支持多线程，提高扫描速度

## 使用方法

1. 克隆仓库到本地：
    ```sh
    git clone https://github.com/xcx2006/dir-blast.git
    ```
2. 进入项目目录：
    ```sh
    cd dir-blast
    ```
3. 运行脚本：
    ```sh
    python main.py
    ```
## 使用私有字典

如果需要使用私有字典，可以将字典文件放置在 `file` 目录下，并命名为 `dictionary.txt`。脚本会自动使用该字典文件进行目录暴力破解。
